# Screenshots

This repo ships fully rendered chart PNGs in `images/plots/`, which is why no
screenshots are strictly required to understand the analysis. If you want to
add polish for recruiters skimming quickly, consider adding:

1. A screenshot of `notebooks/02_exploratory_data_analysis.ipynb` open in
   Jupyter/VS Code, showing a markdown "business question" cell next to its
   code and output.
2. A screenshot of the interactive Plotly treemap
   (`images/plots/15_interactive_country_treemap.html`) open in a browser.
3. A screenshot of the repository's GitHub file listing, showing the clean
   folder structure.

To capture #1 and #2 locally:

```bash
jupyter notebook notebooks/02_exploratory_data_analysis.ipynb
# or open images/plots/15_interactive_country_treemap.html directly in a browser
```

Save any screenshots into this `reports/` folder as `screenshot_01.png`,
`screenshot_02.png`, etc., and reference them from the main `README.md`.
