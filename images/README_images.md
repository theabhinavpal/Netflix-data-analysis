# Chart Index — images/plots/

All charts are generated by `src/visualization.py` from the cleaned dataset.
Re-run `python src/visualization.py` to regenerate every file below.

| File | Chart Type | Description |
|---|---|---|
| `01_movies_vs_tv_bar.png` | Bar | Movies vs. TV Shows title count |
| `02_content_growth_line.png` | Line | Catalog growth by year added, by type |
| `03_top_countries_bar.png` | Horizontal Bar | Top 10 content-producing countries |
| `04_top_directors_bar.png` | Horizontal Bar | Top 10 most prolific directors |
| `05_genre_distribution_bar.png` | Horizontal Bar | Top 12 genres across the catalog |
| `06_rating_distribution_pie.png` | Pie | Content rating distribution (top 7) |
| `07_duration_histogram.png` | Histogram | Movie runtime distribution with KDE |
| `08_duration_boxplot_by_genre.png` | Box Plot | Movie runtime spread across top 8 genres |
| `09_monthly_additions_countplot.png` | Count Plot | Content additions by month (seasonality) |
| `10_correlation_heatmap.png` | Heatmap | Correlation matrix of numeric features |
| `11_content_by_decade_bar.png` | Bar | Titles by release decade |
| `12_seasons_countplot.png` | Count Plot | TV shows by number of seasons |
| `13_scatter_age_vs_duration.png` | Scatter | Content age at addition vs. movie runtime |
| `14_interactive_growth_area.html` | Interactive (Plotly) | Catalog growth area chart |
| `15_interactive_country_treemap.html` | Interactive (Plotly) | Country volume treemap |

Interactive `.html` files can be opened directly in any browser — no server
required.
